# Frontier Exploration with a prior

## Dependency

- Python 3.9

### Check your Python version

```sh
python --version
```

---

## Install project as package

```sh
python -m pip install --no-cache git+https://github.com/ekumenlabs/Frontier-Exploration-with-a-prior.git@master
```

---

## Dev workflow

```sh
source dev.sh
```

---


## When you're done:

```sh
deactivate
```
